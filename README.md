# Daily Support Guide

Daily Support Guide is a neutral education site for everyday routines, label reading, and comparison skills.

Phase 1A scope:
- Static website hosted from GitHub Pages.
- No analytics, affiliate links, coupons, brand recommendations, or retailer links.
- English-first content for a United States audience.
- General education only; no medical advice, diagnosis, treatment, prevention, or product-specific recommendation.

Planned publishing:
- One approved article per day.
- Target release time: 7:00 AM America/New_York.
- Articles remain scheduled until they pass editorial and AI review gates.

Domain:
- Primary: dailysupportguide.com
- Repository: dailysupportguide/dailysupportguide
